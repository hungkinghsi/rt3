Creating our folder structure

We need a folder structure that gives us decent separation between the project's configuration files, source code, and files to distribute to our web server.

All our configuration files will go in our project route:


1. Open Visual Studio Code in an empty folder of your choice, and create a folder called src. This will hold our source code.

2. Let's also create a folder called dist. This will hold the files to distribute to our web server.
