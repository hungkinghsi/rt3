Adding a repository search component‌‌

In this section, we'll add a component to search for a GitHub repository and return some information about it:


1. Let's start by creating a new file called RepoSearch.tsx containing the following imports:


import * as React from "react"; import gql from "graphql-tag";

import { ApolloClient } from "apollo-boost";


2. We are going to take in ApolloClient as a prop. So, let's add an interface for that:


interface IProps {

client: ApolloClient<any>;

}


3. Next, we'll scaffold our component:


const RepoSearch: React.SFC<IProps> = props => { return null;

}


export default RepoSearch;


4. Let's reference this in our App component, now in App.tsx, by first importing it:


import RepoSearch from "./RepoSearch";


5. We can now add this under the app header passing in ApolloClient:


<ApolloProvider client={client}>

<div className="App">

<header className="App-header">

<Header />

</header>

<RepoSearch client={client} />

</div>

</ApolloProvider>
